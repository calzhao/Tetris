## Tetris: The Little Prince



#### 背景介绍

一款基于经典游戏“俄罗斯方块”开发的pygame小游戏。该作品以经典童话《小王子》为给玩家温馨平和的情感体验。在游戏中玩家可自行决定游戏难度，你可以挑战困难模式彰显精湛的俄罗斯方块技术，也可以在简单模式中探索小王子的心路历程。



#### **键盘按键操作**

- 模式选择：点击 `A`（简单模式）, `N`（正常模式）, `H`（困难模式）
- 游戏中：
  - `LEFT` `RIGHT` 控制砖块左右移动
  - `UP` 改变砖块旋转状态
  - `DOWN` 让砖块加速降落
  - `SPACE` 让砖块直接着陆
  - `ESCAPE` 暂停游戏，`R` 结束暂停
